# Security Policy

If you discover a security issue in `INK CHECKER`, please avoid posting sensitive details in a public issue.

For responsible disclosure, contact the maintainer through the repository owner account:

- [snibzyz on GitHub](https://github.com/snibzyz)

When reporting an issue, include:

- affected extension version
- reproduction steps
- potential impact
- screenshots or sample text if relevant

If the report is valid, the issue will be reviewed and fixed in a future release.
